Olá!

Este projeto foi feito pelo Vinícius Arado, para você implementar o projeto em sua máquina será necessário possuir o banco de dados criado no MySql (disponibilizado no arquivo: "proj_cliente_db") e o NetBeans 8.2 instalado na sua máquina.

Pode ser que algumas ações não funcione corretamente devido a várialvel de ambiente.

O projeto foi desenvolvido utilizando a linguagem de programação Java e possui os padrões CRUD, MVC e DAO.


JAVA | JAVASCRIPT | MySQL | HTML | CSS
